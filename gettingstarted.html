Using Maven

Use the parser:

<dependency>
    <groupId>com.github.javaparser</groupId>
    <artifactId>javaparser-core</artifactId>
    <version>3.0.0-RC.2</version>
</dependency>

Use the symbolsolver:

<dependency>
    <groupId>com.github.javaparser</groupId>
    <artifactId>java-symbol-solver-core</artifactId>
    <version>0.4.0</version>
</dependency>

Using Gradle

Use the parser:

compile 'com.github.javaparser:javaparser-core:3.0.0-RC.2'

Use the symbolsolver:

compile 'com.github.javaparser:java-symbol-solver-core:0.4.0'
